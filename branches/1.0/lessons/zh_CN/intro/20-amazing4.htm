20. 神奇-第四部分

你可能已经发现了，当机器人放下beeper后，我们太匆忙就让他前进了。 我们应当首先检查一下是否有墙阻碍他前进。 这里有一个解决方案。

def turn_right():
    repeat(turn_left, 3)

put_beeper()
# begin changes
if not front_is_clear():
    turn_left()
# end changes
move()
while not on_beeper():
    if right_is_clear():
        turn_right()
        move()
    elif front_is_clear():
        move()
    else:
        turn_left()

turn_off()

现在试一下这个程序，可以发现它成功了！ 你可以想象出在哪种情况下它是不能工作的吗？
