Befehlssammlung

Das Erlernen der Computerprogrammierung sollte Spaß machen, Erwachsenen genau so wie Kindern. RUR-PLE ist eine Umgebung, die dazu entwickelt wurde, das Programmieren von Computern unter zuhilfenahme der Sprache Python zu erleichtern. In einer künstlichen Welt in der ein Roboter so programmiert werden kann, dass er verschiedene Aufgaben erledigt, werden Sie lernen, was es heißt, ein Computerprogramm unter Benutzung von Python zu schreiben. Sie haben auch die Möglichkeit, ihre erlernten Fähigkeiten in einer eher konventionellen Umgebung unter Benutzung des eingebauten Interpreters anzuwenden. Auch wenn ihnen dies alles bislang noch nicht viel sagt brauchen sie sich keine Gedanken machen.

Obwohl programmieren lernen Spaß machen kann, müssen sie dennoch ihrerseits Einsatz zeigen. Ich werde ihre Unterstüzung sein, so dass sie alleine lernen können; wie auch immer, solange sie keine eigenen Programme schreiben, werden sie auch nichts lernen. Es ist wichtig, dass sie dies als grundlegende Regel sehen.

Regel # 1
Das Erlernen der Computerprogrammierung ist nicht anders wie das Erlernen eines Musikinstuments: sie müssen es tun und nicht nur darüber lesen.

Die zweite Regel ist das wichtigste und meist gehütete Geheimnis für das Schreiben von Computerprogrammen.

Regel # 2
Schreibe deine Programme so, dass sie für Menschen einfach zu lesen sind.

Ja, richtig, schreiben sie ihre Programme so, dass Menschen wie sie ihre Programme problemlos alleine lesen und verstehen können. Computersprachen sind dazu erstellt worden um mit Computern kommunizieren zu können, genau wie die menschliche Sprache sich dazu entwickelt hat, dass Menschen sich untereinander unterhalten können. Computersprachen sind aber einfacher als menschliche Sprache und werden oft dazu benutzt, dass Programmierer ihre Arbeit mit anderen teilen können. Genau so wie sie ihre Fähigkeiten im Deutsch-Schreiben verbessern wenn sie gut geschriebene Bücher lesen, können sie ihre Programmierkenntnisse ebenso durch gut geschriebene Programme verbessern. Trotzdem müssen sie sie immer noch selber schreiben und zu wissen, wie es geht.

Um mit dem Programmieren zu beginnen werden sie auf dem Bildschirm einem Roboter namens Reeborg verschiedene Aufgaben zuweisen. Während sie dies machen werden sie grundlegende Programmierkonzepte und eine Sprache namens Python lernen. Später werden sie Reeborg verlassen und lernen, wie sie andere Dinge mit dem Computer machen können.
