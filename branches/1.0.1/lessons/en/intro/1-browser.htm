1. About the browser

Rur-ple is organized in a notebook. The first page (or tab) is a very simple web browser and is entitled RUR: Read and Learn. This browser is very limited; it is not recommended that you use it to read the lessons as they often will not display properly.

Rur-ple's browser includes the following four buttons to help you navigate. In likely order of usefulness, they are:

  • always brings you back to the starting page.
  • takes you to the page you were viewing previously, if any.
  • takes you forward to the page you saw after the current one (if any; it is the opposite of the action described above).
  • opens a local file for viewing in the browser window.
  • Finally, is a language selector for RUR-PLE. At present, 7 languages are supported and the lessons are available in English, Chinese and Turkish (85%).
