16. 暂时 ...

当我们想在某个特定条件被满足前重复一些指令时，Python给了我们一个简单的方法来写这段程序，就是用一个新的关键词： while. 首先，我来演示一下如何用pseudocode 来重新写一个我们之前看到的例子。

当不在beeper上，
... 保持前进;
否则,
... 停止.

你会说，这个表达的跟之前的意思一样。 用Python代码，这里我们实际是这样写的:

while not on_beeper():
    move()
 
turn_off()

这里就不再需要重复了！ 试试看！

轮到你了

用 while 和 not重新写一段跨栏程序，这样你就不需要使用那么多的重复了。 换句话说，你程序的核心应该像是这样子的：

while not next_to_a_beeper():
    move_or_jump()
 
turn_off()

保证它是可以运行的！

除草和播种时刻

又是春天了。 乐跑机器人的爸爸已经在花园里播下了种子，等待秋天的收获。 就像上次一样，一些地方两个种子都发芽了，而有些地方一个种子也没发芽。 一个典型的状态如下所示 (file: harvest4.wld).

帮助机器人给花园除草，使得没有一处是有两个胡萝卜（beepers）的，并且在没有胡萝卜的地方重新播种。

这儿有一个 建议 ，部分代码可以使用新关键词 while:

# introducing vocabulary related to the problem
next_to_a_carrot = on_beeper
plant_carrot = put_beeper
pick_carrot = pick_beeper

def one_carrot_only():
    while next_to_a_carrot():
        pick_carrot()      # 把它们全捡起来!            
    plant_carrot()         #重新再种一个!

这段代码与之前的相比短了很多 ( def中的三行取代了之前的六行). 并且，当同一点不止两个种子发芽的情况下，它也是可以运行的。 试试看！
记住，事实上，移除种子再把它们重新正确地播种并 不是一个好主意！
