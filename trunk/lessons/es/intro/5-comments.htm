5. Algún comentario?

Abre el primer programa que esribiste. 

Es muy importante que el programa sea fácil de leer para otras personas. Para esto hay un truco: podemos dejar notas para otras personas que lean el programa (no para el ordenador, ni para Reeborg). A esas notas les llamamos commentarios. Añade este texto en la primera línea de tu programa:

# Mi primer programa

Así que tu programa ahora debe estar así:

# Mi primer programa
move()
move()
turn_off()

La línea que empieza con el símbolo  #, aparece en verde. Este símbolo indica que el resto de la línea es un comentario, que Reeborg ignorará.
