第一部： 欢迎来到Reeborg的世界

在这一部分，我们开始利用Reeborg的世界来探索，从而学到一些简单的编程。

  • 1. 关于浏览器
    第一课简单的介绍一下RUR-PLE的浏览器。
  • 2. Reeborg探索他的世界
    在这一课，您将会比较熟识Reeborg的世界和环境，学习怎么利用键盘的箭头来控制 Reeborg的动作。
  • 3. 第一个程序
    这一课是学习电脑程序，其实就是介绍两个简单操作Reeborg的指令: move() 和 turn_off()。
  • 4. 怎么处理错误
    学习关于Python的灵敏度。
  • 5. 有建议吗？
    我们学习怎么增加一些额外对人类有意义的解释在程序里面，我们将叫它为意见。 我们还会学习怎么去翻译这些用了同义词的“另类”人类语言，让Reeborg跟从的指令，这个跟 变数很类次，在以后我们会更详细介绍。
  • 6. 向左转
    学习操控Reeborg的指令: turn_left()。
  • 7. 报警器
    学习关于报警器和之前两个操控Reeborg的指令的用法: pick_beeper() and put_beeper()。
  • 8. 补丁（有错误）!
    在这一课，我们学习关于错误，还有RUR-PLE怎么帮助我们找到错误和解决方法。
  • 9. 建立墙壁
    我们学习在Reeborg的世界里怎么增加和删除墙壁；这个课程概括了几个编写程序的挑战。

第二部：Reeborg 认识 Python

在第二部分，我们学习一些 Python 的关键词。怎么利用这些关键词来让Reeborg完成越来越难的任务，也让我们更容易地编写程序。

  • 11. 再避免重复！

  • 12. 如果只有Reeborg能够确定...

  • 13. 听我说 ... 如果不是的话 ....
    在这一课，我们学习第三个Python关键词: else.
  • 14. 如果，要不，如果，要不 ....
    在这一课，我们学习第四个，和第五个Python关键词: elif 和 pass。
  • 15. 不是真的！？
    这一课学习第六个Python关键词： not。
  • 16. 一段时间
    这一课学习第七个Python关键词： while。
  • 17. 令人惊异的 - 第一部
    把所有一切学习过的东西都用来解决一些复习，每个都有令人惊讶的总结。
  • 18. 令人惊异的 - 第二部
  • 19. 令人惊异的 - 第三部
  • 20. 令人惊异的 - 第四部
  • 21. 令人惊异的 - 第五部
  • 22. 下雨了！
    帮助Reeborg关闭他家的窗口来避免雨水的进入。
  • 23. 风暴过后
    在风暴过后帮助Reeborg清洁他的大院。
  • 24. 教育Reeborg增加东西
    我们教育Reeborg怎么去增加两个数字。然后我们将会离开Reeborg的世界去进一步学习Phython。

第三部：Python翻译器

在第三部分，我们将会学习Python翻译器，我们也会学习数字，附带条件，表格，价值表（元组），词典。这个部分很像一个传统初学Python的课程 ...但是我们可以利用Reeborg一起学习！

  • 25. Python已经知道怎么增加
    这一课我们探索怎么利用Python来加，减，乘以，除以一些数字。
  • 26. 翻译 Python 的关键词
    这一课我们探索怎么利用一些我们已经知道的关键词来翻译Python。
  • 27. 变数
    这一课我们为大家介绍一个非常重要的理论 变数
  • 28. 变数和功能
    这一课我们一起看一下怎么一起用功能和变数。
  • 29. 您的名字？
    这一课我们学习怎么用变数来代表附带条件。我们还学习限制。
  • 30. 目标为本的编程:  "点" 记数法
    这一课我们学习关于"点" 记数法，这个通常用于目标为本的编程语言上。我们也会学习一些关于Reeborg世界新的事物。
  • 31. 全球化是坏事
    这一课我们学习一个新的Python关键词： global。
  • 32. 很多“返回“
    这一课我们学习另外一个新的Python关键词: return，我们可以写更有用的功能。
  • 33. Repeat() 隐蔽了一些 Python 的关键词
    这一课我们学习两个新的Python关键词：for 和 in。
  • 34. 巧克力奶油冻蛋糕的食谱
    这一课我们学习Python 表，看一下怎么把给Python的目标的所有方法都找出来，学习一些新的关键词， del。 我们更会学一下新的功能：range()。
  • 35. 另外一种的定义
    这一课我们学习Python dicts，和一些内置的功能 str()。
  • 36. 最后一个程序
    这一课我们学习关于Python 元组，易变和不易变目标的分别。
  • 37. 固定Reeborg
    介绍怎么利用现有的东西去制造新的类别。
  • 38. 一些关于类别
    另外两个在Reeborg世界里面的次类别例子。
  • 39. 避免重复 - 很重要
    我们一起学习怎么引入模块。
  • 40. 随机离开
    我们一起看一下两个比较简单的需要依赖随机动作的程序来在迷宫里面走找到出路。第一需要查看前面有没有墙壁，当没有的时候才向前走。例子： 走之前先看。
    第二需要知道一些“例外规测”（除了要看有没有墙壁以外还有其他规测）。例子： 宁可要求别人宽恕也不要容许可。
    这个课程介绍了一些Python的关键词 try， except， 和 raise。
