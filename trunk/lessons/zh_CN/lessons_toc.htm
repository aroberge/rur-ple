学习Python: 儿童利用RUR-PLE玩游戏！

在下面，我们可以看见所有可以选择来学习的课程表，让我们可以由浅入深的学习。

第一部： 欢迎来到乐跑的世界(乐跑机器人的英文名字为Reeborg)

在这一部分，我们开始利用乐跑的世界来探索，从而学到一些简单的编程。

  • 1. 关于浏览器
    第一课简单的介绍一下RUR-PLE的浏览器。
  • 2. 乐跑探索世界
    在第二课你将会比较熟识乐跑的世界和环境，学习怎么利用键盘的箭头来控制 乐跑的动作。
  • 3. 第一个程序
    第三棵让你学习简单的电脑编程，将会向你介绍两个简单操作乐跑的命令: move() （前进）和 turn_off()（关闭）。
  • 4. 出错了怎么办？
    学习关于Python的灵活性。
  • 5. 代码注释？
    我们学习怎么增加一些额外对人类有意义的解释在程序里面，我们将叫它为注释。 我们还会学习怎么去翻译这些用了同义词的“另类”人类语言，让乐跑跟从的命令，这个跟 变数很类次，在以后我们会更详细介绍。
  • 6. 向左转
    学习操控乐跑的命令: turn_left()。
  • 7. 报警器
    学习关于报警器和之前两个操控乐跑的命令的用法: pick_beeper() （捡报警器）和 put_beeper()（放报警器）。
  • 8. 修补错误!
    在这一课，我们学习关于错误，还有RUR-PLE怎么帮助我们找到错误和解决方法。
  • 9. 砌墙
    我们学习在乐跑的世界里怎么增加和删除墙壁；这个课程概括了几个编写程序的挑战。

第二部：乐跑 认识 Python

在第二部分，我们再进一步学习一些新的 Python 关键词。怎么利用这些关键词来让乐跑完成越来越难的任务，也让我们更容易地编写命令和程序。

  • 10. 利用定义来避免重复
    在这个课程，我们学习第一个Python关键词：def。
  • 11. 再一次避免重复
    这个课程，我们学习怎么让乐跑自动地重复一些命令，从而避免反复地输入命令。
  • 12.假如乐跑机器人可以自己决定...
    在这个课程，我们学习第二个Python关键词：if。
  • 13. 听我的 ...否则 ...
    在这一课，我们学习第三个Python关键词: else.
  • 14. 假如，否则，假如，否则 ....
    在这一课，我们学习第四个，和第五个Python关键词: elif 和 pass。
  • 15. 不是真的！？
    这一课学习第六个Python关键词： not。
  • 16. 暂时 ...
    这一课学习第七个Python关键词： while。
  • 17. 神奇的解决方案- 复习一
    把所有一切学习过的东西都用来解决一些复习，每个都有令人惊讶的总结。
  • 18. 神奇的解决方案 - 复习二
  • 19. 神奇的解决方案 - 复习三
  • 20. 神奇的解决方案 - 复习四
  • 21. 神奇的解决方案 - 复习五
  • 22. 下雨了！
    帮助乐跑关闭他家的窗口来避免雨水的进入。
  • 23. 暴风雨过后
    在风暴过后帮助乐跑清洁他的大院。
  • 24. 教会乐跑做加法
    我们教育乐跑怎么去增加两个数字。然后我们将会离开乐跑的世界去进一步学习Python。

第三部：Python翻译器

在第三部分，我们将会学习Python翻译器，我们也会学习数字，附带条件，表格，价值表（元组），词典。这个部分很像一个传统初学Python的课程 ...但是我们可以利用乐跑一起学习！

  • 25. Python已经知道了如何做加法
    这一课我们探索怎么利用Python来加，减，乘以，除以一些数字。
  • 26. 翻译 Python 命令
    这一课我们探索怎么利用一些我们已经知道的命令来翻译Python。
  • 27. 变量
    这一课我们为大家介绍一个非常重要的理论 变量
  • 28. 变量和函数
    这一课我们一起看一下怎么一起用变量和函数。
  • 29. 你叫什么名字？
    这一课我们学习怎么用变数来代表附带条件。我们还学习限制。
  • 30. 面向对象的编程: "点"符号
    这一课我们学习关于"点" 符号，这个通常用于面向对象的编程语言上。我们也会学习一些关于乐跑世界新的事物。
  • 31. 全球化是坏事
    这一课我们学习一个新的Python命令： global。
  • 32. 很多“返回“
    这一课我们学习另外一个新的Python命令: return，我们可以写更有用的功能。
  • 33. Repeat() 隐蔽了一些 Python 的命令
    这一课我们学习两个新的Python命令：for 和 in。
  • 34. 巧克力慕斯蛋糕食谱
    这一课我们学习Python 表，看一下怎么把给Python的目标的所有方法都找出来，学习一些新的命令， del。 我们更会学一下新的功能：range()。
  • 35. 另外一种意义
    这一课我们学习Python dicts，和一些内置的功能 str()。
  • 36. 最后一个序列
    这一课我们学习关于Python 数组，易变和不易变目标的分别。
  • 37. 修补乐跑机器人
    介绍怎么利用现有的东西去制造新的类别。
  • 38. 体验类 "class"
    另外两个在乐跑世界里面的次类别例子。
  • 39. 避免重复 - import
    我们一起学习怎么引入模块。
  • 40. 随机离开
    我们一起看一下两个比较简单的需要依赖随机动作的程序来在迷宫里面走找到出路。第一需要查看前面有没有墙壁，当没有的时候才向前走。例子： 走之前先看。
    第二需要知道一些“例外规测”（除了要看有没有墙壁以外还有其他规测）。这个课程介绍了一些Python的新命令 try（尝试）， except（除了）， 和 raise（提出）。
