29. 你叫什么名字？

我们已经知道了怎么给变量赋值。我们同样也可以给字符串赋值，例如：

>>> FirstName = "Andre"
>>> LastName = "Roberge"
>>> FullName = FirstName + LastName
>>> print FullName
AndreRoberge

注意Python是怎么用加号(+)把两个字符串连结到一起了，这个操作我们成为字符串连结。注意一下名和姓中间没有空格的。我们可以通过在两个字符串中间插入第三个由单个空格组成的字符串来更正这个名字。

>>> FullName = FirstName + " " + LastName
>>> print FullName
Andre Roberge

还有一种方法可以得到相同的结果，就是用我们用过的 %s ，来把多个变量放到一个字符串内。

>>> name = "%s %s" % (FirstName, LastName)
>>> print name
Andre Roberge

最后，我们可以比较两个字符串是否相等。

>>> print name == FullName
True

分割

有时候我们仅仅需要使用字符串里的单个字符。为此，我们需要学习分割。

我会给你们三条规则，一条关于表示方法的提示，和一系列例子的结果。这些例子你可以自己用Python解释器来尝试。除了这三条规则，我不会 给你任何别的解释，因此你需要自己仔细观察来完全理解发生了什么。

  1. Python从0开始计数。
  2. 在计数的时候，python不会包括你所期待的最后的数字。
  3. Python能按一定步长计数。
  • 有用的表达方法： [first:last:step]
    ;三个都是可选的。

这儿有一些例子。

>>> alphabet = "abcdefghijklmnopqrstuvwxyz"
>>> digits = '0123456789'

>>> alphabet[0]
'a'

>>> alphabet[1]
'b'

>>> alphabet[5]
'f'

>>> digits[0]
'0'

>>> digits[7]
'7'

>>> digits[-1]
'9'

>>> digits[-2]
'8'

>>> alphabet[-3]
'x'

>>> digits[0:3]
'012'

>>> digits[0:4]
'0123'

>>> digits[0:5]
'01234'

>>> digits[2:5]
'234'

>>> alphabet[2:5]
'cde'

>>> digits[0:10:2]
'02468'

>>> digits[0::2]
'02468'

>>> digits[0::3]
'0369'

>>> digits[::3]
'0369'

>>> digits[::-1]
'9876543210'

>>> digits[::-2]
'97531'

>>> digits[::-3]
'9630'

>>> alphabet[:10:2]
'acegi'
