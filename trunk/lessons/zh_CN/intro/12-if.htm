第一个决定

说实话，机器人做决定时确实需要一些帮助：你必须给他一些选择，他才能做决定。 例如，当机器人在一个报警器的右边，所以在屏幕上看起来他就在这个报警器上，你可以给他一些做法的选择：你可以叫他把它捡起来：

if on_beeper():
    pick_beeper()

让我们来看看上述代码的意思：

  • Python 的关键词 if告诉了机器人某个 条件, 这个条件是真 或者 假是要被后面的程序用到的。
  • 当机器人在一个报警器上（屏幕上显示是机器是跟报警器在同一个位置）on_beeper()是个值为 真的条件，而情况相反时，就为 假
    [注意，在之前rur-ple的版本中，这个命令被称为 next_to_a_beeper()]
  • 冒号 (:) 必须放在 当 条件是 真实的情况下机器人要遵循的指令的前面。
  • 如同我们在定义时一样，这一系列的命令是已经预先设定好的。

这段解释读起来好像很复杂，但事实上使用 if 命令是很简单的. 让我们在一个简单例子中看看。 假设我们希望机器人走9步，并且要把所有沿途的报警器都捡起来。 （假设在每一点上最多只有一个报警器。）例如，开始的时候，乐跑的位置可能如下所示：

我们希望乐跑最后的位置在:

所以，我们希望机器人要做的是：

  • 向前走一步；
  • 检查是否有报警器；
  • 如果有一个报警器，则将它捡起来；否则继续前进；

重复以上的步骤9次。 记住，当地上没有报警器而我们命令机器人执行捡这个动作时，他就会停下来不再继续执行命令。 如下是我们要做的：

def move_and_pick():
    move()
    if on_beeper():
        pick_beeper()

repeat(move_and_pick, 9)
turn_off()
