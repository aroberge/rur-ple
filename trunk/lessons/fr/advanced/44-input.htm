Reeborg vous interroge

Vous pouvez donner à Reeborg quelques entrées en utilisant ce qui suit :

input_string()
input_int()

Ces fonctions retournent une valeur qui peut être affectée à une variable. Elles peuvent aussi  être utilisées avec une chaîne additionnelle... 

input_string("Quel est votre nom ?")
input_int("Quel est votre nombre favori ?")
