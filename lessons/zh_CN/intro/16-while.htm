16. 暂时 ...

当我们想在某个特定条件被满足前重复一些指令时，Python给了我们一个简单的方法来写这段程序，就是用一个新的关键词： while。首先，我来演示一下如何用虚拟码 来重新写一个我们之前看到的例子。

当不在报警器上，
... 保持前进;
否则,
... 停止.

你会说，这个表达的跟之前的意思一样。 用Python代码，这里我们实际是这样写的:

while not on_beeper():
    move()
 
turn_off()

这里就不再需要重复了！ 试试看吧！

轮到你了

用 while 和 not重新写一段跨栏的程序，这样你就不需要使用那么多重复的命令了。换句话说，你程序的核心应该像是这样子的：

while not on_beeper():
    move_or_jump()
 
turn_off()

你要确保程序是可以运行的，别放弃哦！

小实验：除草和播种时刻

又是春天了。 乐跑的爸爸已经在花园里播下了种子，等待秋天的收获。 就像上次一样，一些地方两个种子都发芽了，而有些地方一个种子也没发芽。 一个典型的例子如下图所示 (见文档：harvest4.wld)。

现在大家一起来帮助乐跑给花园除草，使得没有一个地方有两个胡萝卜（这里我们用报警器来表示），而在没有胡萝卜的地方重新播种，目的是要让每个地方都有一个胡萝卜。

这儿有一个 建议 ，部分代码可以使用新关键词 while:

# 介绍跟问题有关的单词
next_to_a_carrot = on_beeper
plant_carrot = put_beeper
pick_carrot = pick_beeper

def one_carrot_only():
    while next_to_a_carrot():
        pick_carrot()      # 把它们全捡起来!            
    plant_carrot()         #重新再种一个!

这段代码与之前的相比短了很多吧？ ( def中的三行取代了之前的六行)。并且，当同一点不止两个种子发芽的情况下，乐跑也是可以继续运行的。
试试看！
记住，事实上，移除种子再把它们重新正确地播种并 不是一个好主意！
